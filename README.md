
# Jobeezy

Jobeezy is a full-stack job portal website built with modern technologies. Users can browse jobs, apply to positions, track applications in a dashboard, and interact with an AI chatbot for career advice.


## Features
- Browse and search jobs by title and type
- AI Chatbot by GitHub Models (openai/gpt-4.1)

- Post new jobs with full form


- Dashboard to view and manage applied jobs
- Apply to jobs with "Applied" state tracking
- Clerk authentication
